资料库使用文档

本网站由于为了让大家可以更好的永久使用，我们经过日思夜想（站长头发都秃了），经过层层考验，突破各种技术壁垒，现推出以下下载方案

下载VPN（如果现有VPN工具可以跳过）

  1. 点击购买机场

  2. 下载VPN客户端

    • windows/mac 推荐使用点击下载适合自己系统的版本
    • Android 推荐使用点击下载适合自己系统的版本

  3. 安装好VPN软件之后 以window举例

    • 打开安装好的clash软件点击进入IOS VPN奶妈教程
  4. 安装好VPN软件之后 以Android举例
    • 打开安装好的clash软件点击进入安卓VPN奶妈教程
  5. IOS VPN APP都是收费的暂时不推荐

下载Telegram（如果现有Telegram工具可以跳过）

点击进入TG奶妈教程

畅想书站之旅Come

开始下载文件

  1. 找到自己想要的书籍点击获取资源

  2. 等待返回结果，返回成功之后点击**复制跳转**

  3. 连接会自动跳转到Telegram 的资源页面，浏览器会默认弹出打开外部应用提醒，然后点击 **打开Telegram Desktop**
  4. 进入到Telegram 聊天群，在窗口粘贴复制内容，然后等待book_root机器人自动返回结果

  5. 接收到返回结果在文件上面右键点击，选择保存文件到本地
